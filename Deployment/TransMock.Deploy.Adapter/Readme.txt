This is the NuGet package for the TransMock adapter for BizTalk Server. 

It contains the assemblies for the adapter for the following BizTalk Server versions:
- BizTalk Server 2010
- BizTalk Server 2013
- BizTalk Server 2013 R2
- BizTalk Server 2016
